Route commands


        sudo apt-get install net-tools 
        route 
        route -n 
        sudo route add default gw 169.254.0.0
        sudo route add -host 192.168.1.51 reject
        ip route 
        route del default
        sudo route add -net default gw 10.10.0.1 netmask 0.0.0.0 dev wlan0 metric 1
        sudo route del -net default gw 10.10.0.1 netmask 0.0.0.0 dev wlan0 metric 0

        
route flags

        U - route is up
        G - specified gateway
        D - Dynamically allocated
        M - modified
        R - reinstalled
        ! - rejecting route
